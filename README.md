# Система управления личной библиотекой

Консольное приложение для управления личной библиотекой, реализующее основные принципы объектно-ориентированного программирования на C++.

## Функциональность
- Управление каталогом книг (добавление, поиск, отображение)
- Регистрация пользователей и выдача книг
- Автоматическая загрузка/сохранение данных в файл
- Валидация ввода и обработка ошибок
- Интерактивное консольное меню

## Требования
- Компилятор C++ с поддержкой стандарта C++17
- CMake 3.10 или выше

## Сборка и запуск

# Сборка проекта
```
mkdir build
cd build
cmake -G "MinGW Makefiles" ..
mingw32-make
```
# Запуск программы
```
./library
```
